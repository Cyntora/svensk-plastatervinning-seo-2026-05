# Svensk Plaståtervinning · SEO Action Plan · maj 2026

Single-file HTML-rapport hostad på GitHub Pages.

Riktad SEO-genomgång och prioriterad åtgärdsplan för svenskplastatervinning.se — levererad av Cyntora 21 maj 2026.

## Filer

- `index.html` — scroll-deck-rapport, single-file med inbäddad CSS
- `slides.pdf` — 16:9 PDF-backup (1600 × 900)
- `README.md` — den här filen

## Visa lokalt

Öppna `index.html` direkt i en browser. Inget byggsteg.

## Tekniska val

- Cormorant Garamond för rubriker, Inter för brödtext
- Scroll-deck-layout, varje sektion fyller en viewport
- `<meta name="robots" content="noindex, nofollow">` så Google inte indexerar innehållet
- Cyntoras logotyp som inline-SVG
